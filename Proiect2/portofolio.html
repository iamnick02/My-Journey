Project 1 - LAN Party Simulation

This project was created during my studies in algorithm design. The goal was to simulate a LAN party using concepts like stacks, queues, lists, and trees. Below are the key details:

  • Language: C
  • Description: The program organizes and simulates the progress of a LAN party, managing teams and games in a hierarchical manner. Each team competes, and the winners advance until a final winner is declared.
  • Data Structures Used:
    • Stacks
    • Queues
    • Linked Lists
    • Binary Trees
  • Challenges: Implementing efficient memory management for dynamic data structures and ensuring the correctness of the game logic.
  • Outcome: Successfully simulated a LAN Party with accurate team management and results tracking.
